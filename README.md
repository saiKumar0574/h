 🎮 Hangman Game

A simple console-based Hangman game built using Python.

## 📌 Features

- 🎲 Random word selection
- 🔤 Letter-by-letter guessing
- ❌ Maximum 6 incorrect guesses
- ✅ Input validation
- 🏆 Win/Lose conditions
- 💻 Console-based gameplay

## 🛠️ Technologies Used

- Python
- Random Module
- While Loop
- If-Else Statements
- Lists
- Strings
